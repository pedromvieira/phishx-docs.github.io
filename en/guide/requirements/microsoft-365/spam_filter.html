# Spam filter

# TL;DR

  1. Create a new Spam Filter in protection -> spam filter. Add our current domains list: Domains

# Settings

  1. Access the current Microsoft 365 Exchange admin center.

    Exchange Admin Center

  1. Go to spam filter in menu: protection -> spam filter.

    Add a new one. Click +.

    Reference

  1. In new spam filter policy -> *Name fill with "PhishX".

  1. In new spam filter policy -> Domains allow list -> Always deliver email from the following domains to the inbox.

    Click +.

  2. In add allowed domain add our current domains list: Domains.

    Click Ok.

TIP

Remove any extra spaces after paste.

  1. In new spam filter policy -> Applied To select The recipient domain is and add your own domains.

    Click Ok.

  1. Click Save.
