# Connectors

# TL;DR

  1. Create a New Connector in Mail flow -> Connectors. Add our current IPs list: IPs

# Settings

  1. Access the preview Microsoft 365 Exchange admin center.

    Exchange Admin Center

  2. Go to Connectors in menu: Mail flow -> Connectors.

    Reference

    Click Add a connector.

  1. In New Connector -> Connection from select Partner organization.

    Click Next.

  1. In Connector name -> Name fill with "PhishX" and select Turn it on.

    Click Next.

  1. In Authenticating sent email select By verifying that the IP address of the sending server matches one of the following IP addresses, which belong to your partner organization. Add our current IPs list: IPs.

    Click Next.

  1. In Security restrictions select Reject email messages if they aren't sent over TLS.

    Click Next.

  1. In Review Connector.

    Click Create connector.

  1. In Connector created.

    Click Done.
